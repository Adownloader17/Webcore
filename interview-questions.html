HTML Interview Questions

1. What is HTML?

HTML (HyperText Markup Language) is used to structure content on the web.

2. Difference between tag and element?

A tag is a keyword inside angle brackets, while an element includes the opening tag, content, and closing tag.

3. What are semantic HTML tags?

Semantic tags describe the meaning of content, such as <header>, <nav>, <section>, and <footer>.

4. Difference between div and span?

<div> is a block-level element, while <span> is an inline element.

Git & GitHub Interview Questions

5. What is Git?

Git is a version control system that tracks changes in source code.

6. What is GitHub?

GitHub is a platform used to host Git repositories and collaborate with other developers.

7. Difference between Git and GitHub?

Git is a tool for version control, while GitHub is a web service that hosts Git repositories.

Linux Interview Questions

8. Why is Linux used by developers?

Linux is widely used because it is stable, secure, and commonly used on servers.

9. What does the ls command do?

The ls command lists files and folders in a directory.

SEO & Accessibility Questions

10. What is SEO?

SEO (Search Engine Optimization) helps websites rank better in search engines.

11. Why is alt text important?

Alt text helps screen readers and improves SEO when images fail to load.

12. What is web accessibility?

Web accessibility ensures that websites can be used by people with disabilities.
