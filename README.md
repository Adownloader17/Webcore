# WebCore - Frontend Foundations

WebCore is a comprehensive learning platform for frontend developers. It covers fundamental web technologies including HTML, CSS, JavaScript, Git, Linux, SEO, and web accessibility. This project is designed to help students build strong foundations in frontend development.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation** - Smooth scrolling and intuitive navigation
- **Comprehensive Guides** - In-depth coverage of HTML, CSS, JavaScript, Git, Linux, SEO, and Accessibility
- **Practical Examples** - Real-world code examples and best practices
- **Programming Books Library** - Access to 12 categorized programming books with direct PDF links
- **SEO Optimized** - Built with semantic HTML and modern SEO practices
- **Accessible** - WCAG compliant with proper accessibility features
- **Professional UI** - Modern design with gradient headers, smooth animations, and scroll-to-top button
- **Interview Questions** - Comprehensive collection of frontend interview questions
