Semantic Tags

Tag Purpose
<header> Page or section header
<nav> Navigation links
<main> Main content
<article> Independent content
<section> Thematic grouping

Common Elements

Element Usage
<h1> - <h6> Headings (h1 should be used once per page)
<p> Paragraphs
<a href=""> Hyperlinks
<img src="" alt=""> Images (always use alt text)
<ul>, <ol> Unordered and ordered lists
<form> User input forms

üé® CSS Guide

Styling

Selectors

Selector Example Description
Element p { } Selects all <p> tags
Class .btn { } Selects elements with class "btn"
ID #header { } Selects element with id "header"
Attribute [type="text"] { } Selects by attribute value
Pseudo-class a:hover { } Selects element in specific state

Box Model

* { margin: 0; /* Outside spacing */ padding: 0; /* Inside spacing */ border: 1px; /* Border thickness */ box-sizing: border-box; /* Include padding/border in width */ }

Flexbox Layout

.container { display: flex; justify-content: center; /* Horizontal alignment */ align-items: center; /* Vertical alignment */ gap: 20px; /* Space between items */ }
